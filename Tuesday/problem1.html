b tag

Use the b keywords in summary, product names in a review, or other spans of text whose typical presentation would be boldfaced. The b tag element was meant to make text boldface.

i tag

The element is used to display the words in italic type.

u tag

It is used to annotating spelling errors,u tag should not be used to underline text for presentation purposes.

select tag

This tag helps in getting one or more option for the question asked as in multiple choice.

button

This tag is used to create a button which can perform certain task assigned to it.

hr tag

This tag will divide the content which is above and below of this tag with a horizontal line dividing it.

br tag

This tag is used for line break in a block of text.
